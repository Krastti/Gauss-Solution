# Сравнение методов решения СЛАУ

Проект реализует лабораторную работу по сравнению методов решения систем линейных алгебраических уравнений:

- метод Гаусса без выбора ведущего элемента;
- метод Гаусса с частичным выбором главного элемента по столбцу;
- LU-разложение без перестановок;
- решение треугольных систем прямой и обратной подстановкой.

## Структура проекта

- `src/gauss` — метод Гаусса, прямой и обратный ход, подстановки, невязка `||Ax - b||`.
- `src/lu` — LU-разложение, решение через `Ly = b` и `Ux = y`.
- `src/matgen` — генерация случайных матриц, случайных векторов, матриц Гильберта, правой части по точному решению.
- `src/experiments` — запуск лабораторных экспериментов и запись CSV-отчётов.
- `src/timer` — высокоточный таймер.
- `src/logging` — простой логгер.
- `src/main.c` — консольный интерфейс для ручного решения СЛАУ и запуска экспериментов.
- `data/reports` — отчёты и CSV-файлы с результатами экспериментов.

Программа сохраняет результаты в CSV-файлы:

- `data/reports/single_system_experiment.csv` — сравнение времени решения одной системы для разных `n`;
- `data/reports/multiple_rhs_experiment.csv` — сравнение метода Гаусса с LU при нескольких правых частях;
- `data/reports/hilbert_accuracy_experiment.csv` — проверка точности на матрицах Гильберта.

## Сборка и запуск

Для сборки нужен CMake 4.0+ и C-компилятор, доступный из командной строки.

1. Сконфигурировать проект:

```bash
cmake -S . -B build
```

2. Собрать приложение:

```bash
cmake --build build
```

3. Запустить приложение на Linux/macOS или при single-config генераторе:

```bash
./build/GaussSolution
```

На Windows при single-config генераторе:

```powershell
.\build\GaussSolution.exe
```

На Windows с генератором Visual Studio исполняемый файл может находиться в подпапке конфигурации:

```powershell
.\build\Debug\GaussSolution.exe
```

После запуска программа показывает меню. В нём можно ввести свою систему линейных алгебраических уравнений, выбрать метод решения или запустить лабораторные эксперименты с сохранением CSV-файлов.
